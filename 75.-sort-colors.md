````java
```java
public void sortColors(int[] a) {
    int i = 0;  // index of 0
    int j = 0;  // processing ele 
    int k = a.length - 1;  // index of 2

    while (j <= k) {
        if (a[j] == 0) {
            swap(a, i++, j++);
        } else if (a[j] == 2) {
            swap(a, j, k--);
        } else {
            j++;
        }
    }
}

private void swap(int[] a, int i, int j) {
    int tmp = a[i];
    a[i] = a[j];
    a[j] = tmp;
}
```
````

#### TC & SC:&#x20;

> TC: O(n)
>
> SC: O(1)
