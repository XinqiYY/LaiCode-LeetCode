## 解法一

> Clarification:&#x20;
>
> Algorithm: 使用bitwise，当数字不为0时，去除末尾1

#### <mark style="color:red;">注意：</mark>

## 代码

```java
public class Solution {
    // you need to treat n as an unsigned value
    public int hammingWeight(int n) {
        int sum = 0;
        while (n != 0) {
            sum++;
            // 最快消除末尾1的方法
            n = n & (n - 1);
        }
        
        return sum;
    }
}
```

#### TC & SC:&#x20;

> TC: O(1)
>
> SC: O(1)
