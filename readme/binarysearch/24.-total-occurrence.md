## 代码

```java
public int totalOccurrence(int[] a, int t) {
  if (a == null || a.length == 0) return 0;
  if (a.length == 1 && a[0] == t) return 1;
  
  int l = 0, r = a.length - 1;
  while (l < r - 1) {
    int m = l + (r - l) / 2;
    if (t <= a[m]) {
      r = m;
    } else {
      l = m;
    }
  }

  int res = 0;
  while (l + 1 <= a.length && a[l] == t) {
    res++;
    l++;
  }

  while (r + 1 <= a.length && a[r] == t) {
    res++;
    r++;
  }

  return res;
}
```

#### TC & SC:&#x20;

> TC: O(logn) worst O(n)
>
> SC: O(1)
