## 代码

```java
public ListNode selectionSort(ListNode head) {
  ListNode cur = head;
  while (cur != null) {
    ListNode min = cur;
    ListNode rest = cur.next;
    while (rest != null) {
      if (rest.value < min.value) {
        min = rest;
      }
      rest = rest.next;
    }

    int tmp = cur.value;
    cur.value = min.value;
    min.value = tmp;
    cur = cur.next;
  }
  
  return head;
}
```

#### TC & SC:&#x20;

> TC: O(n^2)
>
> SC: O(1)
