## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> i 和 j 同时向后走，\
> i 用来标记 0的index，j用来判断当前ele是否是 0
>
> 如果不是0，则swap，i++
>
> 如果是0，j++

#### <mark style="color:red;">注意：</mark>

## 代码

````java
```java
class Solution {
    public void moveZeroes(int[] nums) {
        int j = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                int temp = nums[i];
                nums[i] = nums[j];
                nums[j] = temp;
                j++;
            }
        }
    }
}
```
````

#### TC & SC:&#x20;

> TC: O(n)
>
> SC: O(1)
