## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> * base case：如果到底了，返回0
> * 向左右两边孩子高度，取最大+1

#### <mark style="color:red;">注意：</mark>

## 代码

```java
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;
        
        int left = maxDepth(root.left);
        int right = maxDepth(root.right);
        
        return Math.max(left, right) + 1;
    }
}
```

#### TC & SC:&#x20;

> TC: O(n) # of nodes
>
> SC: O(height)
