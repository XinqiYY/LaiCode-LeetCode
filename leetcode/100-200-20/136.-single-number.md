## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> * ‘^’ 符号在bit中，相同为0，不同为1，有点像对对碰，利用它的特性就可以得出落单的是谁

#### <mark style="color:red;">注意：</mark>

## 代码

````java
```java
public int singleNumber(int[] nums) {
    int res = 0;
    for (int n : nums) {
        res ^= n;
    }
    return res;
}
```
````

#### TC & SC:&#x20;

> TC: O(n)
>
> SC: O(1)
