> * 找到第一小，再找到第二小，如果出现比第二小大的，则返回true

## 代码

````java
```java
public boolean increasingTriplet(int[] nums) {
    int min = Integer.MAX_VALUE;
    int secMin = Integer.MAX_VALUE;

    for (int n : nums) {
        if (n <= min) min = n;
        else if (n < secMin) secMin = n;
        else if (n > secMin) return true;
    }

    return false;
}
```
````

#### TC & SC:&#x20;

> TC: O(n)
>
> SC: O(1)
