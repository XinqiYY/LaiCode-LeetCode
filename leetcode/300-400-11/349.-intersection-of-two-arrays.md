## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> * setA存储nums1的ele，遍历nums2，如果ele在setA中存在，则放入setB，最后遍历setB，把elements放到res中

## 代码

````java
```java
public int[] intersection(int[] nums1, int[] nums2) {
    Set<Integer> s1 = new HashSet<>();
    for (int n : nums1) {
        s1.add(n);
    }
    
    Set<Integer> s2 = new HashSet<>();
    for (int n : nums2) {
        if (s1.contains(n)) {
            s2.add(n);
        }
    }

    int[] res = new int[s2.size()];
    int i = 0;
    for (int n : s2) {
        res[i++] = n;
    }

    return res;
}
```
````

#### TC & SC:&#x20;

> TC: O(n)
>
> SC: O(n)
