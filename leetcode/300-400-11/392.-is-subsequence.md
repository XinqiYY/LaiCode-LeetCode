## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> * subsequence是从前往后，不能是乱序
> * 所以指针 i, j 分别指向两个string，如果两个char相等i++,j++, else j++

#### <mark style="color:red;">注意：</mark>

## 代码

````java
```java
public boolean isSubsequence(String s, String t) {
    int i = 0, j = 0;
    
    while (i < s.length() && j < t.length()) {
        if (s.charAt(i) == t.charAt(j)) i++;
        j++;
    }

    return i == s.length();
}
```
````

#### TC & SC:&#x20;

> TC: O(max(n,m))
>
> SC: O(1)
