## 代码

```sql
select p.project_id, round(avg(e.experience_years), 2) as average_years
from Project p
left join Employee e
on p.employee_id = e.employee_id;
group by p.project_id; # 根据project_id计算平均值，如果不加这行则计算了所有平均值
```
