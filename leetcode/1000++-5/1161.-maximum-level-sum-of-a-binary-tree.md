## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> * BFS遍历，计算每一层node的合，如果有比max大的则更新max以及记录哪一层是max

## 代码

````java
```java
public int maxLevelSum(TreeNode root) {
    int max = Integer.MIN_VALUE;
    int lev = 0, maxLev = 0;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while (!q.isEmpty()) {
        int size = q.size();
        int cur = 0;
        lev++;
        for (int i = 0; i < size; i++) {
            TreeNode node = q.poll();
            cur += node.val;

            if (node.left != null) {
                q.offer(node.left);
            }

            if (node.right != null) {
                q.offer(node.right);
            }
        }

        if (cur > max) {
            max = cur;
            maxLev = lev;
        }
        
    }

    return maxLev;
}
```
````

> SC: O(d)
