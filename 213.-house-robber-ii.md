## 代码

```java
class Solution {
    public int rob(int[] nums) {
        if (nums.length == 1) return nums[0];
        return Math.max(helper(0, nums.length - 2, nums), helper(1, nums.length - 1, nums));
    }
    
    private int helper(int lo, int hi, int[] nums) {
        int rob1 = 0;
        int rob2 = 0;
        for (int i = lo; i <= hi; i++) {
            int tmp = Math.max(rob1 + nums[i], rob2);
            rob1 = rob2;
            rob2 = tmp;
        }
        
        return rob2;
    }
}
```

#### TC & SC:&#x20;

> TC: O(n)
>
> SC: O(1)
