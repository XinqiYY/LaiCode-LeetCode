## 解法一

> Clarification: 不能使用加减法
>
> Algorithm:&#x20;
>
> * 使用Bitwise operation
> * 临时值C，存储a\&b
> * a = a ^ b
> * b =(c) << 1

#### <mark style="color:red;">注意：</mark>

## 代码

```java
class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int res = a ^ b;
            int car = (a & b) << 1;
            a = res;
            b = car;
        }
        
        return a;
    }
}
```

#### TC & SC:&#x20;

1. TC: O(1)
2. SC: O(1)
