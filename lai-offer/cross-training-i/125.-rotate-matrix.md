## 代码

```java
public class Solution {
  public void rotate(int[][] matrix) {
    int top = 0;
    int down = matrix.length - 1;
    while (top < down) {
      int[] tmp = matrix[top];
      matrix[top] = matrix[down];
      matrix[down] = tmp;
      top++;
      down--;
    }

    for (int i = 0; i < matrix.length; i++) {
      for (int j = i + 1; j < matrix[i].length; j++) {
        int tmp = matrix[i][j];
        matrix[i][j] = matrix[j][i];
        matrix[j][i] = tmp;
      }
    }
  }
}

```

#### TC & SC:&#x20;

1. TC: O(m\*n)
2. SC: O(1)
