## 代码

````java
```java
public List<Integer> maxWindows(int[] a, int k) {
  List<Integer> res = new ArrayList<>();
  if (a == null || a.length == 0) return res;

  for (int i = 0; i <= a.length - k; i++) {
    int max = a[i];
    for (int j = i + k - 1; j >= i; j--) {
      if (a[j] > max) {
        max = a[j];
      }
    }
    res.add(max);
  }

  return res;
}
```
````
