## 解法一

> Clarification:&#x20;
>
> Algorithm:&#x20;
>
> * sort一遍，指针分别指向两头，相等的话count++，移动i,j, else 比k小挪 i，比k大挪 j

#### <mark style="color:red;">注意：</mark>

## 代码

````java
```java
public int maxOperations(int[] a, int k) {
    Arrays.sort(a);
    int count = 0;
    int i = 0, j = a.length - 1;

    while (i < j) {
        if (a[i] + a[j] == k) {
            count++;
            i++;
            j--;
        } else if (a[i] + a[j] < k) {
            i++;
        } else {
            j--;
        }
    }

    return count;
}
```
````

#### TC & SC:&#x20;

> TC: O(nlogn)
>
> SC: O(logn)
