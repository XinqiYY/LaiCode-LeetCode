## 代码

````java
```java
public List<Double> averageOfLevels(TreeNode root) {
    if (root == null) return null;

    List<Double> res = new ArrayList<>();
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while (!q.isEmpty()) {
        int size = q.size();
        double sum = 0;
        double ave = 0;
        for (int i = 0; i < size; i++) {
            TreeNode cur = q.poll();
            sum += cur.val;
            if (cur.left != null) {
                q.offer(cur.left);
            }

            if (cur.right != null) {
                q.offer(cur.right);
            }
        }

        ave = sum / size;
        res.add(ave);
    }

    return res;
}
```
````

> TC: O(n, # of node)
>
> SC: O(n)
